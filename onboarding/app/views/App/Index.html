Welcome to the Cloud Native Computing Team

The purpose of this service is to automate the preparation of a new CNCT employee's technical onboarding, as a list of tasks, represented as issues in GitHub.

Instructions

  1. Click the below Authorize button to first login into your Github account and allow this service to create tasks on your behalf. Then this service will:
  2. Create a Milestone and a Project in GitHub.
  3. Create Issues in Github to represent onboarding tasks, and link them to the Milestone and Project.
  4. Assigns those issues to your Github user.
